# Agent Customization

This workspace uses `copilot-instructions.md` to provide guidance for GitHub Copilot and similar AI assistants.

## Agent Behavior

- Use the project context from `copilot-instructions.md`.
- Prefer small, safe, Expo-compatible code changes.
- Preserve the existing structure and style of the app.
- Prioritize TypeScript and React Native/Expo idioms.

## Purpose

Help contributors make changes to this Expo Router app in a consistent and compatible way.

## Notes

- The workspace is a single Expo app, not a monorepo.
- The primary focus is mobile app UI and navigation within the Expo managed workflow.
